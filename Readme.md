# 💖 Birthday Surprise Web Experience

A small interactive web experience crafted with love, animations, and a little bit of magic ✨
Made as a special birthday gift.

---

## 🌸 About The Project

This is a multi-step interactive web page designed to create an emotional and memorable experience.

It includes:

* Smooth transitions between sections
* Animated UI elements
* 3D floating hearts using Three.js
* A final celebration moment with confetti 🎉

Every screen is designed to reveal feelings step by step, like unwrapping a gift.

---

## 🚀 Tech Stack

* HTML5
* Tailwind CSS
* JavaScript
* Three.js (for 3D animations)
* GSAP (for smooth animations)
* Canvas Confetti

---

## 📸 Features

* 💌 Step-by-step emotional storytelling
* 💖 Animated heart elements in 3D space
* 📷 Polaroid-style memory section
* 🎊 Confetti celebration finale
* 📱 Responsive design (works on mobile & desktop)
